# Tab Palette

Obsidian用の高度なタブ管理プラグインです。

## 機能

- タブ一覧をパレット形式で表示
- 除外フォルダの設定（特定のフォルダのタブを非表示）
- タグとディレクトリパスを1行に表示
- キーボードショートカットでタブ操作
  - 矢印キー: タブ選択の移動
  - Enter: タブを開く
  - w: タブを閉じる
  - p: タブをピン/アンピン
- 並び順の選択（履歴順 / 開いた順）
- 常に新しいタブで開く設定
- 前後のタブに移動するコマンド

## 使い方

1. コマンドパレット（Ctrl/Cmd + P）から「タブパレットを開く」を実行
2. 矢印キーでタブを選択
3. Enter でタブを開く、w で閉じる、p でピン

## 設定

プラグイン設定から以下を設定できます：

- 除外フォルダ
- タグ表示の有無
- パス表示の有無
- 並び順
- 常に新しいタブで開く

## キーボードショートカット

設定から以下のコマンドにショートカットを割り当てられます：

- タブパレットを開く
- 前のタブに移動
- 次のタブに移動
